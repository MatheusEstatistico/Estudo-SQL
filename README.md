# Estudo-SQL
Repositório dedicado ao estudo da linguagem SQL para fins estatísticos, de ciência de dados e engenharia de dados. 

## Curso 
É possível aprender a usar o SQL através do curso construido pelo [Téo](https://github.com/TeoCalvo/SQL-Estat), do canal [Téo Me Why](https://www.youtube.com/@teomewhy). O curso é usando o VSCode e o SQLite.
- Playlist do curso: https://www.youtube.com/watch?v=VmkJG8awKqM&list=PLvlkVRRKOYFRo651oD0JptVqfQGDvMi3j&index=1 
- Banco de dados usado no curso: https://www.kaggle.com/datasets/teocalvo/teomewhy-loyalty-system
